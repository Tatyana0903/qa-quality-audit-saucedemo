Jira Project Management Overview
--
The testing process was managed using Jira following a Scrum-based workflow.

Project structure:
--
- Epic – SauceDemo Functional Testing
- Stories – Login, Cart, Checkout
- Tasks – Test Cases (TC-XXX format)
- Bugs – Defects identified during execution
- Sprint – SCRUM Sprint 1

Each defect is linked to:
--
- Related Test Case
- Related Story

Related Risk from the Risk Register

This ensures full traceability between requirements, risks, test cases and defects.
