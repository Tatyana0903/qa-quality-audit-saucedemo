## Application Overview
SauceDemo is a demo e-commerce web application that allows users to browse products,
add them to a cart, and complete a checkout flow.

## In Scope
The following areas are included in the quality audit:
- User authentication (login and logout)
- Product catalog and product list
- Sorting functionality
- Cart management (add/remove items)
- Checkout process (without payment)

## Out of Scope
The following areas are excluded from the audit:
- Performance and load testing
- Security and penetration testing
- Backend services and APIs
- Payment processing

## Scope Rationale
The audit focuses on core user flows that have the highest impact on user experience
and potential business value.
